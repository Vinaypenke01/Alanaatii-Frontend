# Alanaatii – Personalized Letters, Boxes & Gifts

Alanaatii is a premium e-commerce platform dedicated to the art of personalized communication and thoughtful gifting. We help users express their deepest emotions through handcrafted letters, elegant boxes, and curated gift sets.

## 🌟 Overview

Alanaatii provides a seamless experience for creating and sending personalized gifts. Whether it's a heartfelt love letter, a birthday message, or a luxury gift box, our platform ensures every detail is crafted with care and delivered with magic.

## ✨ Key Features

- **Personalized Order Flow**: A step-by-step process to choose products, customize content, and select packaging.
- **Product Variety**:
  - **Scripts**: Professionally written content for any occasion.
  - **Letter Papers**: A selection of premium, floral, and textured papers.
  - **Letters**: Handcrafted or typed letters with customizable styles (Telugu, English, Handwritten).
  - **Boxes**: Elegant packaging ranging from basic to luxury satin-lined boxes.
  - **Gifts**: Curated additions like teddy bears, chocolates, and flowers.
- **Dedicated Dashboards**:
  - **User Dashboard**: Track orders, manage profiles, and view personal scripts.
  - **Writer Dashboard**: A specialized interface for content creators to manage requests and revisions.
  - **Admin Panel**: Comprehensive management of products, pricing, orders, and site analytics.
- **Modern UI/UX**: Built with a focus on aesthetics, featuring smooth transitions, gradients, and responsive design.

## 🚀 Tech Stack

- **Frontend**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/) + [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) + [Shadcn UI](https://ui.shadcn.com/)
- **State Management**: [Zustand](https://github.com/pmndrs/zustand) (Order Store)
- **Data Fetching**: [TanStack Query](https://tanstack.com/query/latest)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Routing**: [React Router](https://reactrouter.com/)

## 🛠️ Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   bun install
   ```

3. Start the development server:
   ```bash
   npm run dev
   # or
   bun run dev
   ```

4. Open [http://localhost:8080](http://localhost:8080) in your browser.

## 📂 Project Structure

- `src/components`: Reusable UI components and layout elements.
- `src/pages`: Main application pages (Home, Products, Order Flow, Dashboards).
- `src/lib`: Utility functions, store definitions, and data constants.
- `src/hooks`: Custom React hooks for shared logic.
- `public`: Static assets like icons and images.
